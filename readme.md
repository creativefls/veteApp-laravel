## How to Run

1. Install node module
```
npm install
```

2. Install laravel
```
composer install
```

3. Generate CSS and JS to minify version
```
npm run production
```

4. Migrate the database
```
php artisan migrate:refresh --seed
```
5. And run your local server
```
php artisan serve
```
